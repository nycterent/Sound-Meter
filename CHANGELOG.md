# Changelog

All notable changes to this project will be documented in this file.

## [1.4.1] - 2026-02-18

### Added
- Sound level measurement in decibels
- Real-time sound level visualization with speedometer UI
- Chart view for sound level history
- Webhook integration for sending sound level data to external services
- Webhook settings configuration screen
- Multi-language support (English, French, Chinese)
- GitHub Actions workflow for building APK
- GitHub Actions workflow for automated releases

### Features
- Measures ambient sound level using device microphone
- Displays current sound level in dB
- Visual gauge showing sound intensity
- Historical data tracking with chart
- Configurable webhook notifications
- Minimum API level: Android 5.0 (API 21)
- Target API level: Android 14 (API 34)
